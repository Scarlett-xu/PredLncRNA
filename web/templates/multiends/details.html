Models

The summary of computational methods used for the lncRNA prediction
Type Algorithm Method Description Source code available Input format Training species
Binary classifier based SVM iSeeRNA A lightweight SVM-based program to predict long intergenic noncoding RNAs (lincRNAs) √ GTF, GFF, BED Human, mouse
CNCI A signature tool effectively analyzing adjacent nucleotide triads and distinguishing the protein-coding and noncoding sequences independent of known annotations √ FASTA, GTF Human, plant
PLEK A predictor, which uses an improved computing pipeline based on k-mer and SVM to distinguish between lncRNAs and mRNAs √ FASTA Human, maize
LncRScan-SVM An SVM-based method integrating characteristics from a genetic structure, transcript sequence, potential codon sequence and conservatism √ GTF Human
CPC An SVM-based classifier, Coding Potential Calculator (CPC), predicting lncRNAs based on six sequence features × FASTA Eukaryotic
CPC2 A faster and more accurate version of CPC √ FASTA, GTF Human, mouse, fly,zebrafish
Longdist An SVM-based approach using the 1st ORF relative length and frequencies of nucleotide patterns selected by PCA as features √ FASTA Human, mouse, zebrafish
CPPred A method using the SVM algorithm and multiple sequence features to predict lncRNAs √ FASTA Human, mouse
LGC An algorithm to predict lncRNAs in a wide range of species based on the relationship between ORF length and GC content √ FASTA, BED, GTF Human, plant
LR CPAT A nonalignment method is based on a logical regression model by extracting sequence characteristics √ FASTA, BED Human, mouse, fly,zebrafish
lncScore An alignment-free tool using a logistic regression model with 11 selected features √ GTF, GFF Human, mouse
RF LncRNA-ID An identification method based on Random Forest and three features including ORF, ribosome interaction and protein conservation √ FSATA, BED Human, mouse
LncRNApred A method to differentiate between lncRNAs and protein-coding transcripts based on RF × FASTA Human
FEELnc An RF-based method providing a standardized and detailed protocol for identifying and annotating lncRNAs √ GTF Human, mouse, dog
COME A tool for calculating the coding potential based on a balanced random forest model √ GTF Human, mouse, fly, worm, plant
Deep learning based DSN lncRNA-MFDL A DSN-based classifier using multiple features √ FASTA Human
DNN DeepLNC A method based on Shannon entropy and DNN × FASTA Human
RNN mRNN A gated RNN-based method which can interpret context and long-range information dependencies √ FASTA Human
RNN, CNN lncRNAnet A deep learning method using RNN to learn the intrinsic behavior of lncRNAs and CNN to detect stop codons to obtain an ORF indicator √ FASTA Human, mouse
DBN, DNN LncADeep An identification tool using both DBN and DNN √ FASTA Human, mouse
Ensemble learning based Ensemble Learning lncFinder An integrated lncRNA identification package that constructed classifiers based on different categories of features and various machine learning algorithms √ FASTA Human, mouse, wheat
PredLnc-GFStack An identification tool based on a combination of genetic algorithm and stacked ensemble learning √ FASTA Human, mouse
LncPred-IEL An lncRNA prediction method using iterative ensemble learning √ FASTA Human, mouse
CPE-SLDI A coding potential estimation approach using oversampling technology to solve the local data imbalance √ FASTA Human
