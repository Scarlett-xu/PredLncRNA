OUR WORK

We introduce seven major databases with lncRNA resources, which are commonly used in lncRNA prediction.

We classify all features into six categories: codonrelated features, ORF-related features, GC-related features, transcript-related features, coding sequence related features and structure-related features, and detailed descriptions and formulas of features are provided.

We review all lncRNA prediction methods and divide them into three categories. The details of computational methods are described and analyzed.

We provide a python package named ezLncPred to integrate existing state-of-the-art lncRNA prediction methods. Regardless of the runtime environment requirements of methods, users can apply different methods easily with ezLncPred.
